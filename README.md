# 🎨 Personal Portfolio with React

Uma Single Page Application (SPA) moderna desenvolvida com **React** apresentando meu portfólio profissional com animações fluidas, design responsivo e navegação dinâmica.

---

## 🌟 Funcionalidades

- **SPA Responsiva** com navegação fluida entre seções
- **Animações Interativas** com Framer Motion para melhor UX
- **Rotas Dinâmicas** para navegação otimizada
- **Design Responsivo** adaptável a todos os dispositivos
- **Acessibilidade** seguindo padrões WCAG
- **Performance Otimizada** com lazy loading e otimizações de código
- **SEO Friendly** com meta tags e estrutura semântica
- **Tema Moderno** com CSS Modules para estilização isolada

---

## 🚀 Tecnologias Utilizadas

### Frontend Core
- **React 18** - Library principal para construção da UI
- **React Router DOM** - Navegação e roteamento dinâmico
- **JavaScript ES6+** - Linguagem de programação moderna

### Styling & Animation
- **CSS3 & CSS Modules** - Estilização modular e isolada
- **Framer Motion** - Animações fluidas e interativas
- **Responsive Design** - Mobile-first approach
- **Flexbox & Grid** - Layout moderno e flexível

### Development Tools
- **Vite** - Build tool rápido para desenvolvimento
- **ESLint** - Linting para qualidade de código
- **Prettier** - Formatação consistente de código

### Performance & SEO
- **React.lazy()** - Code splitting para otimização
- **Semantic HTML** - Estrutura acessível e SEO friendly
- **Meta Tags** - Otimização para redes sociais e busca

## 💻 Como Rodar Localmente

1. **Clone o repositório:**
```bash
git clone https://github.com/ovitelas/react-portfolio.git
cd react-portfolio
```

2. **Instale as dependências:**
```bash
npm install
# ou
yarn install
```

3. **Execute o projeto em desenvolvimento:**
```bash
npm run dev
# ou
yarn dev
```

4. **Acesse no navegador:**
```
http://localhost:3000
```

5. **Build para produção:**
```bash
npm run build
# ou
yarn build
```
